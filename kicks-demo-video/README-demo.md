# Kicks Store Demo Video

Archivo renderizado:

```text
renders/kicks-store-demo.mp4
```

Duracion verificada por FFmpeg: `00:02:30.00`.

## Comandos utiles

Preview en HyperFrames Studio:

```powershell
cd kicks-demo-video
npm run dev
```

Validar composicion:

```powershell
npm run check
```

Renderizar usando FFmpeg local incluido en el proyecto:

```powershell
npm run render:demo
```

## Notas

- El video no incluye voz para que puedas grabar la narracion despues.
- Usa `voiceover-script.md` como base de narracion.
- El video es una composicion HyperFrames estilizada, no una captura directa del navegador.
