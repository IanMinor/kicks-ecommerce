# Guion de voz para Kicks Store Demo

Duracion sugerida: 2:20 a 2:30.

> Este proyecto es Kicks Store, un ecommerce de sneakers desarrollado con React, Vite, Tailwind, Zustand, Express y MySQL.

> La pagina principal presenta una experiencia visual moderna, con un hero llamativo, productos destacados y accesos rapidos por categoria.

> En el catalogo, el usuario puede explorar productos con filtros dinamicos por genero, talla, color, categoria y precio. La experiencia se mantiene fluida al actualizar resultados.

> Cada producto cuenta con una vista detallada donde se muestran precio, descripcion, color y talla disponible. Desde aqui se puede agregar al carrito o iniciar la compra directamente.

> El carrito se sincroniza en tiempo real desde la barra de navegacion y tambien permite revisar productos desde la pagina completa del carrito.

> En el checkout, el usuario completa los datos de envio, revisa el resumen del pedido y confirma la compra.

> Luego se muestra una pantalla de confirmacion con estado del pedido, total y detalles principales.

> Finalmente, la cuenta del usuario funciona como dashboard, mostrando informacion personal y accesos rapidos para continuar comprando o revisar el carrito.

> El resultado es una experiencia ecommerce completa, responsive y visualmente atractiva.
